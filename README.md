# 🧭 Step By Step Guide

## 🧼 Clean-Up & Simple Run

Clean up the cache & install the dependencies

- `yarn clean && yarn start`
- `yarn || npm i`
- `yarn pod` (iOS Only)
- `yarn clean && yarn start`
- `yarn ios`
